# 健康分析功能 - 快速开始

## 🎯 一分钟了解

访谈系统现在可以**自动生成 AI 健康分析报告**！

完成健康咨询访谈后，系统会：
1. ✅ 自动调用 AI 分析所有问答
2. ✅ 生成专业的健康评估报告
3. ✅ 提供个性化改进建议
4. ✅ 保存为 JSON 和文本两种格式

## 🚀 立即使用

### 第一步：设置 API Key

```bash
export STEPFUN_API_KEY='your-api-key-here'
```

### 第二步：运行访谈

```bash
cd /Users/town/code4/questionAgent
python interview_client_hybrid.py
```

### 第三步：查看报告

访谈完成后，报告自动生成在：

```bash
sessions/[最新session]/health_report.txt
```

## 📋 报告包含什么？

- 🏥 **整体健康评分** (0-100)
- ⚠️  **主要健康关注点**
- 🌟 **生活方式评估**（睡眠、运动、饮食、压力）
- 🚨 **风险因素识别**
- 💡 **3-5条改进建议**
- 🏥 **就医指导建议**

## 📁 生成的文件

```
sessions/20251119_HHMMSS/
├── session.json           # 原始访谈数据
├── summary.txt           # 访谈摘要
├── health_analysis.json  # ✨ AI 分析（JSON）
└── health_report.txt     # ✨ 健康报告（文本）
```

## 🧪 验证集成

```bash
python test_health_analysis.py
```

应该看到所有项目都标记为 ✅

## 📖 详细文档

- **完整指南**：`HEALTH_ANALYSIS.md`
- **集成总结**：`INTEGRATION_SUMMARY.md`
- **混合模式**：`README_HYBRID.md`

## 💡 示例报告

```
======================================================================
📋 健康访谈分析报告
======================================================================

🏥 整体健康状况: ⚠️ FAIR
📈 健康评分: 65/100

⚠️  主要健康关注点
   1. 睡眠质量不佳，每晚仅 5-6 小时
   2. 缺乏定期运动
   3. 工作压力较大

💡 健康改进建议
   1. 建立规律作息，争取每晚 7-8 小时睡眠
   2. 每周至少 3 次，每次 30 分钟中等强度运动
   3. 学习压力管理技巧
   ...
```

## ⚠️ 注意事项

- 需要网络连接（调用 Step API）
- 会消耗 API token（约 1000-2000 tokens/次）
- 报告仅供参考，不构成医疗建议
